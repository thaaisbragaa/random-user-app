# Random User Generator

App em React que consome a API https://randomuser.me para gerar perfis aleatórios.

## Funcionalidades

- Gerar usuário aleatório
- Exibir detalhes (foto, nome, idade, email, telefone, localização)
- Adicionar/remover favoritos (salvo no localStorage)
- Dark/Light mode
- Loading e erro básico

## Tecnologias

- React + Vite
- Tailwind CSS
- fetch API
